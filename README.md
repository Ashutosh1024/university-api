# University Student Registration API  
A backend system for managing student course registrations.  
Built using **Node.js**, **Express**, and **MongoDB (Mongoose)**.  
Deployed live on **Render** with database hosted on **MongoDB Atlas**.

### Student Management  
- Add a new student  
- View all students  
- View a student by ID  
- Update student details  
- Delete a student  
- Search students by course  

### Technology Stack
- **Node.js**
- **Express.js**
- **MongoDB Atlas**
- **Mongoose**
- **Render (Deployment)**  
- **Postman (API Testing)**
